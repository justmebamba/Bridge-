# **App Name**: TikTok Bridge

## Core Features:

- TikTok Username Input: Collect the user's TikTok username and send it to the control panel.
- Terms and Conditions: Display and require acceptance of terms regarding third-party monetization solutions for users in countries without native TikTok monetization.
- Control Panel: A subdomain control panel accessible by the admin to view collected TikTok usernames.
- US Number Selection: Present a list of US phone numbers for the user to choose from, which will be linked to their account.
- Number Verification: Looks like a code was sent to the chosen phone number.

## Style Guidelines:

- Primary color: Vibrant purple (#9C27B0) to convey innovation and connection to TikTok's branding.
- Background color: Light gray (#EEEEEE) to ensure content clarity.
- Accent color: Electric blue (#2962FF) to highlight interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', a sans-serif for clear and modern text.
- Use clean, simple icons for form fields and informational sections to aid user comprehension.
- A single-column layout with clear, step-by-step instructions to guide users through the monetization linking process.
- Subtle loading animations and progress indicators to enhance the user experience during verification processes.
